# Contributing

Issue reports, device compatibility results, academic references, and reproducible camera measurements are welcome.

Code contributions are not accepted automatically at this stage. ParticlePair uses a source-available noncommercial license and may also be offered under separate commercial terms. Accepting third-party code without an explicit contributor agreement could create unclear relicensing rights.

Before preparing a substantial pull request, open an issue describing the change and wait for maintainer guidance. Documentation corrections and small test cases may still be accepted after their licensing is confirmed.

When reporting camera results, include the display model, refresh rate, phone/camera model, browser, distance, ambient lighting, modulation strength, and success/error rate. Do not include personal pairing secrets, credentials, or private recordings.
